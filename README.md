# Full-Stack-Booking-System
A full-stack web application that allows users to browse experts, view available time slots, and book sessions in real-time.
Built using **React (Frontend)** and **Node.js + Express (Backend)**.

---

## Features

-  Expert listing with categories
- Time-slot based booking system
- Real-time slot selection
- Prevents double booking
- Booking form with user details
- Auto-refresh slot updates (polling)
- REST API backend

---

## Tech Stack

### Frontend
- React.js
- React Router
- Fetch API

### Backend
- Node.js
- Express.js
- In-memory data storage (for demo)
